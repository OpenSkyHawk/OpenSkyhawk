

Deprecated List



#### Member [**CAN\_ID\_ECHO\_1**](CANProtocol_8h.md#variable-can_id_echo_1)  

Use canIdEcho(1) = 0x301. Wrong base (0x210 vs 0x300).  
Use canIdEcho(2) = 0x302. Wrong base.  
Use canIdEvt(1) = 0x201. Off-by-one.  
Use canIdEvt(2) = 0x202. Off-by-one.  
Use canIdHb(1) = 0x101. Off-by-one: this is canIdHb(0) ([**PanelBridge**](namespacePanelBridge.md) reserved).  
Use canIdHb(2) = 0x102. Off-by-one.  
Use CTRL\_ID\_TEST\_SEQ. Kept for backward compatibility with prototype code.  
Use [**CANProtocol::busOff()**](namespaceCANProtocol.md#function-busoff) instead.  
Use [**CANProtocol::send()**](namespaceCANProtocol.md#function-send) instead.  
Use [**CANProtocol::start()**](namespaceCANProtocol.md#function-start) instead.  
Use [**CANProtocol::rec()**](namespaceCANProtocol.md#function-rec) instead.  
Use [**CANProtocol::tec()**](namespaceCANProtocol.md#function-tec) instead.
