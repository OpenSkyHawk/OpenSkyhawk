## Detailed Description


[**PinRef.h**](PinRef_8h.md) forward-declares `class  ADS1115`. A typedef alias cannot be forward-declared with `class`, so this file defines a concrete wrapper class that inherits every constructor and method from Adafruit\_ADS1115 without adding any logic.


Adafruit\_ADS1115 takes address and bus via begin(), not the constructor. Pattern: [**ADS1115**](classADS1115.md) adc; PanelGroup::registerADC(adc, 0x48, Wire);
