## About OpenSkyhawk { #about }

**Open tools. Real engineering. Shared progress.**

We believe simulation hardware should be accessible to every enthusiast, not locked behind expensive proprietary products. OpenSkyhawk produces fully documented 3D-printed panels, custom PCBs, and STM32 firmware to replicate the full A-4E cockpit experience.

Every component is designed to be reproducible with off-the-shelf parts, standard PCB fabrication services, and a consumer 3D printer. Schematics, firmware, and CAD files are all in the repository.

[Learn More →](https://github.com/OpenSkyHawk/OpenSkyhawk){ .md-button }
