OpenSkyhawk is an open-source initiative dedicated to developing high-quality, modular cockpit hardware for the DCS A-4E Skyhawk. Built by the community, for the community.

We believe simulation hardware should be accessible to every enthusiast, not locked behind expensive proprietary products. OpenSkyhawk produces fully documented 3D-printed panels, custom PCBs, and STM32 firmware to replicate the full A-4E cockpit experience.

Every component is designed to be reproducible with off-the-shelf parts, standard PCB fabrication services, and a consumer 3D printer. The schematics, firmware, and CAD files are all available in the repository.
