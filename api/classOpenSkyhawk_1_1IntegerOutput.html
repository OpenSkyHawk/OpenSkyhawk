Detailed Description

Call an arbitrary function with the raw value from a ControlPacket.

Escape hatch for non-standard output logic — e.g. a value that maps to motor positions, PWM brightness, or multi-bit state machines that do not fit the LED pattern. Mirrors DcsBios::IntegerBuffer.

void onCanopyPos(uint16_t v) { analogWrite(CANOPY_MOTOR, v >> 8); }
