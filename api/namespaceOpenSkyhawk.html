Detailed Description

Output and input classes for OpenSkyhawk panel boards.

Objects are declared at global scope in a sketch; constructors self-register into static linked lists. PanelGroup::loop() dispatches incoming ControlPacket CAN frames to all registered OutputBase objects and polls all registered InputBase objects every iteration.
