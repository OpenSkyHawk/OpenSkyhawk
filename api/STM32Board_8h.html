Detailed Description

Shared STM32F103 hardware initialisation for OpenSkyhawk avionics nodes.

Manages the bi-color status LED, DiagSerial, and CAN peripheral configuration — identical on every OpenSkyhawk STM32F103CBT6 board. All CAN bus operations (send, filter, start) go through CANProtocol.

Fixed hardware (same on every board — no constructor arguments):

  • LED : PB14 (red) + PB15 (green), active HIGH
  • UART : USART1 PA9 TX / PA10 RX @ 115200 (diagnostic tap)
  • CAN : SN65HVD230 on PA11 (RX) / PA12 (TX)
