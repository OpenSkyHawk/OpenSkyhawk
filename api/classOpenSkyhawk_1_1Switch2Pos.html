Detailed Description

Debounced 2-position GPIO switch — sends a ControlPacket CAN event on change.

Mirrors DcsBios::Switch2Pos but sends a CAN packet via PanelGroup::sendEvent() instead of writing to DCS-BIOS directly. The matching OpenSkyhawk::DCSInput on the SimGateway translates the packet to a DCS-BIOS message.

Value sent: 1 when pin is LOW (switch closed to GND), 0 when HIGH.
