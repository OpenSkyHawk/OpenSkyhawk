CAN bus health states reported to STM32Board via onCanStatus().

Enumerator
STARTING 

CAN peripheral configured but not yet started.

NORMAL 

Bus active, no errors.

TX_ERROR 

TEC > 0 — transmit errors accumulating.

BUS_OFF 

CAN controller halted — bus-off condition.
