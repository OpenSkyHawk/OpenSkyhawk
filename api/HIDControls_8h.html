Detailed Description

CAN controlId constants for HID axes and buttons.

Shared between STM32 (via CANProtocol) and RP2040 (SimGateway sketches). Contains only #define constants — no classes, no functions, no state.

controlId routing by range: 0x0010–0x001F HID axes — routed to Joystick axis setters on SimGateway 0x0020–0x009F HID buttons — routed to Joystick button setters on SimGateway
